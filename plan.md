对话历史列表的第一行是一个“创建新对话”的按钮，该按钮为浅蓝色方形小圆角按钮，点击该按钮后会在对话历史列表中创建一个新的对话记录，并且在当前对话展示区域显示一个新的空白对话界面，用户可以在该界面中输入内容来和 AI 进行对话，如果用户已经输入了内容并且和 AI 进行了对话，那么在对话历史列表中该对话记录就会显示出来，并且显示对话的名字（如果用户没有修改过对话名字的话默认调用大模型 API 来生成一个对话名字）和生成时间，用户可以点击该对话记录来查看该对话的内容。

对话历史列表展示用户之前保存的对话记录（用户使用 AI 助手保存的对话记录和AI对话页面中有的对话记录一起存储下来），对话历史列表中的每个对话记录都显示对话的名字和生成时间，用户可以点击对话记录来查看该对话的内容。

右侧为对话显示区域，如果点击左侧的历史对话记录的话右侧会显示对应的对话内容，如果点击了“创建新对话”按钮的话右侧会显示一个新的空白对话界面。对话内容展示区域中，每一个对话都用小圆角的方形边框围起来，并且有一个小箭头指向说话的人（用户的对话框箭头指向右边，AI 的对话框箭头指向左边），聊天内容能够够支持 markdown 和数学公式的渲染。设计要简洁现代，去除掉过于复杂的卡片设计。

右侧底部为输入区域，输入区域有一些预设的选项可以进行选择，例如“我在学习过程中遇到了哪些薄弱点？”、“我应该优先学习哪些知识点？”、“我有哪些题目是错的？”等，也可以自己上传一些 skill 的 md 文档，点击选项后会将对应的问题或者 skill 作为用户发送给 AI 的消息的一部分。输入区域的文本框要求有一个明显的边框，边框颜色可以稍微淡一点，边框四个角用小圆角处理。文本框旁边有一个方形小圆角发送按钮，点击发送按钮后会将用户输入的问题以及之前通过选项选择的内容一起发送给 AI，并且在聊天内容显示区域显示出来（如果上传了 skill 文件，请显示文件而不是文件内容）。设计要简洁现代，去除掉过于复杂的卡片设计。（每个对话历史点开后，都可以继续进行对话）
